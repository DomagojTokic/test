This project is built and tested using Maven. To build it, go to project directory in command line and run command "mvn package". To run tests, run command "mvn test".

After building project, you'll get a JAR file called AntimaTest-1.0.jar in target directory.

To run this program use command "java -jar AltimaTest-1.0.jar *path-to-input-file*". Example of input file is given in "input.txt" file.
